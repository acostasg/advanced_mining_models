Principal Component Analysis (PCA)¶

L'Anlisi de components principals (PCA) aplicada a aquestes dades identifica la combinaci d'atributs (components principals a l'espai de caracterstiques) que expliquen la major variaci de les dades. Aqu dibuixem les diferents mostres dels 2 primers components principals.

L'algorime realiza la mitjana de cada atribut, calcula la matriu de covarianza de les dades, despres calcula els valors i vectors propis de la matriu de covarianza i ordena els factor y projeccins de les dades en l'espai PCA.

Linear discriminant Analysis (LDA)¶

L'anlisi discriminant (LDA) s una tcnica per tractar de trobar les variables que permeten diferenciar als grups i quantes d'aquestes variables sn necessries per aconseguir la millor classificaci possible. La pertinena als grups, coneguda per endavant, s'utilitza com a variable depenent (una variable categrica amb tants valors discrets com a grups). Les variables en les quals suposem que es diferencien els grups s'utilitzen com a variables independents o variables de classificaci (variables discriminants).

L'objectiu de l'anlisi discriminant s trobar la combinaci lineal de les variables discriminants que millor permeti diferenciar (discriminar) als grups. Una vegada trobada aquesta combinaci (la funci discriminant) podr ser utilitzada per classificar nous casos. Es tracta d'una tcnica d'anlisi multivariante capa d'aprofitar les relacions existents entre una gran quantitat de variables independents per maximitzar la capacitat de discriminaci.

Conclucions¶

Tant LDA com PCA sn tcniques de transformaci lineal que s'utilitzen per a la reducci de dimensionalidad. PCA es pot descriure com un algorisme no supervisat, ja que ignora les etiquetes de classe sent el seu objectiu trobar les adreces (components principals) que maximitzen la varincia d'un conjunt de dades. En contrast amb PCA, LDA s supervisat i calcula les adreces (discriminants lineals) que representaran als eixos que maximitzen la separaci entre mltiples classes.

L'amplria i la longitud dels ptals estan altament correlacions i la seva variabilitat a travs de les tres espcies Iris s'explica principalment pel component 1 (0.92461621), que tamb explica una gran part de la variabilitat en la longitud del spal.

Els grafics mostren les espcies versicolor i virginica estan separades en LDA, mentre que hi ha certa superposici entre elles en PCA. PCA tracta de retenir la major part de la variabilitat en les dades mentre que LDA tracta de retenir la major part de la varincia entre la classe de les dades.

def comparacio_amb_svm( X ):
    y = iris.target

    h = .02  # mida de pas a la malla

    # creem una instncia de SVM i ajustem les dades. No escalem el nostres
    # dades ja que volem dibuixar els vectors de suport
    C = 1.0  # SVM regularization parameter
    svc = svm.SVC(kernel='linear', C=C).fit(X, y)
    rbf_svc = svm.SVC(kernel='rbf', gamma=0.7, C=C).fit(X, y)
    poly_svc = svm.SVC(kernel='poly', degree=3, C=C).fit(X, y)
    lin_svc = svm.LinearSVC(C=C).fit(X, y)

    # creeu una malla per dibuixar
    x_min, x_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    y_min, y_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx, yy = np.meshgrid(np.arange(x_min, x_max, h),
                         np.arange(y_min, y_max, h))

    # ttol de les trames
    titles = ['SVC amb nulci lineal',
              'LinearSVC (nulci lineal)',
              'SVC amb nucli RBF',
              'SVC amb nucli polinmic (grau 3)']


    for i, clf in enumerate((svc, lin_svc, rbf_svc, poly_svc)):
        # Traceu el lmit de la decisi. Per aix, assignarem un color a cadascun d'ells
        # punt a la malla [x_min, m_max] x [y_min, y_max].
        plt.subplot(2, 2, i + 1)
        plt.subplots_adjust(wspace=0.4, hspace=0.4)

        Z = clf.predict(np.c_[xx.ravel(), yy.ravel()])

        # Posar el resultat en una trama de colors
        Z = Z.reshape(xx.shape)
        plt.contourf(xx, yy, Z, cmap=plt.cm.Paired, alpha=0.8)

        # Parcella tamb els punts de formaci
        plt.scatter(X[:, 0], X[:, 1], c=y, cmap=plt.cm.Paired)
        plt.xlabel('Sepal length')
        plt.ylabel('Sepal width')
        plt.xlim(xx.min(), xx.max())
        plt.ylim(yy.min(), yy.max())
        plt.xticks(())
        plt.yticks(())
        plt.title(titles[i])

    plt.show()
